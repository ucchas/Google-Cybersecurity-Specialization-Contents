Security risk assessment report

Part 1: Select up to three hardening tools and methods to implement

  1. Impletement password policies
  2. Firewall maintenance and Port filtering
  3. Implementation of MFA (Multi-Factor Authentication)

Part 2: Explain your recommendations

  1. From the inspection, we can find that the admin password was set to default and the organization’s employees’ also share passwords among themselves. These activities can pose serious vulnerabilities to the organization’s network. Hence, the implementation of password policies according to the NIST CSF will be highly effective to tackle future attacks.
  2. It was observed that the firewall is not regularly maintained. It was found that there was no filtering of data traffic going in and out of the firewall which is a huge vulnerability to the network. The implementation of port filtering on the organization’s network can be used to control the traffic on the network. It will be helpful to prevent any types of attacks i.e. DDoS attacks.
  3. While the basic first layer protection using passwords is used, there needs to be another layer of protection for the organization’s network. A Multi-Factor Authentication(MFA) in place can ensure prevention against brute force attacks. It will also help in identity management greatly.
